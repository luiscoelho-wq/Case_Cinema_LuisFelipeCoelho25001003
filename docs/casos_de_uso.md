## 1. Caso de Uso: Agendar Sessão  

**Ator:** Funcionário / Administrador   
**Objetivo:** Criar um novo horário de filme na sala.  

**Passo a Passo:**
1. O funcionário escolhe o **Filme**, a **Sala** e a **Hora**.
2. O sistema checa se a sala está vazia naquele horário.
3. Se estiver livre, o sistema salva o agendamento.
4. Se estiver ocupada, o sistema avisa do erro e não salva.

**Resultado:** Sessão cadastrada com sucesso.
